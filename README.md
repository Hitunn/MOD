## 主要更新 (2026.04.23)
- ✅ **多线程架构**：所有串口通信移至子线程，界面永不卡顿。
- ✅ **定时轮询**：可自由设置轮询间隔（100ms ~ 10s），自动周期性读取设备数据。
- ✅ **动态参数配置**：起始地址（十六进制输入）和寄存器数量（1-125）均可由用户指定。
- ✅ **健壮性增强**：分包粘包处理、超时重试、错误提示。
- ✅ **心跳检测（待完善）**：连续失败计数，为离线判断预留接口。

## 历史版本
- `basic`: 单线程基础版，硬编码读写参数。
- `advanced`: 增加多线程和轮询框架（此分支基于 advanced 开发）。
- `advanced-2.0` (当前): 正式加入可配置轮询、动态参数、完整多线程。

## 技术栈
- Qt 5.11.1 (MinGW)
- C++11
- Modbus RTU 协议 (03/06 功能码)

## 使用说明
1. 选择串口号和波特率，点击“打开串口”。
2. 手动读写：
   - 读：输入起始地址（十六进制，如 `0`）和寄存器数量（1-125），点击“读寄存器”。
   - 写：输入寄存器地址（十六进制）和值（十进制），点击“写入寄存器”。
3. 定时轮询：
   - 设置轮询间隔，点击“开始轮询”，程序按周期自动读取并更新界面。
   - 点击“停止轮询”结束自动读取。
4. 数据会实时显示在表格中（序号、数值、时间）。

## 编译运行
- 使用 Qt Creator 打开 `ModbusMaster.pro`，选择 MinGW 编译器，构建运行。
- 需要虚拟串口工具（如 VSPD）配合 Modbus Slave 软件进行测试。

## 文件结构
- `mainwindow.h/cpp`：主窗口（UI 与信号转发）。
- `modbusworker.h/cpp`：多线程工作类，负责串口通信与协议解析。
- `main.cpp`：程序入口。
