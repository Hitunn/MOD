 Modbus RTU 主站模拟器

一个基于 Qt/C++ 的 Modbus RTU 主站模拟器，支持多从站轮询、请求队列、独立心跳探测与离线降速，适用于工业自动化上位机开发。

## 功能特点

- **Modbus RTU 协议**：完整实现 03（读保持寄存器）和 06（写单个寄存器）功能码，CRC 校验准确。
- **多线程架构**：串口通信工作线程与 UI 线程分离，界面永不卡顿。
- **请求队列**：所有请求（轮询、心跳、手动）串行化排队，避免总线冲突。
- **多从站支持**：可配置从站列表，轮询自动切换地址，心跳独立探测每个从站。
- **心跳与离线探测**：每个从站独立心跳定时器，超时后自动切换慢速探测频率，并标记离线；轮询自动跳过离线从站。
- **超时重试**：心跳和手动请求支持重试（最大 3 次），轮询超时直接放弃。
- **参数可配置**：从站地址、寄存器地址、寄存器数量、轮询间隔等均可通过 UI 设置。
- **日志输出**：关键步骤输出调试信息，便于问题追踪。

## 技术栈

- **语言**：C++11
- **框架**：Qt 5.11（QSerialPort, QTimer, QThread, QQueue, QMap, QList 等）
- **协议**：Modbus RTU
- **构建**：qmake / Qt Creator

## 使用说明

### 环境要求

- Windows/Linux 系统
- Qt 5.11 及以上（MinGW 或 MSVC 编译器）
- 虚拟串口软件（如 VSPD）或真实串口设备

### 编译运行

1. 用 Qt Creator 打开 `seiralport.pro`（项目文件）。
2. 选择合适编译器（如 MinGW 32-bit），构建并运行。
3. 打开串口之前，先配置好从站模拟器（如 Modbus Slave），确保地址、波特率等参数与程序一致。

### 界面操作

- **串口设置**：选择串口号、波特率，点击“打开”。
- **轮询**：设置轮询间隔（ms），点击“开始定时发送”即可自动循环读取所有在线从站的寄存器 0。
- **手动读取**：填写从站地址、寄存器地址、数量，点击“读取寄存器”。
- **手动写入**：填写从站地址、寄存器地址、写入值（十进制），点击“写入寄存器”。
- **状态指示**：界面右上角指示灯显示总线是否至少有一个从站在线；控制台输出每个从站的心跳状态和离线信息。

## 项目结构

- `main.cpp`：程序入口
- `mainwindow.h/cpp`：主界面 UI 及交互逻辑
- `modbusworker.h/cpp`：Modbus 工作线程，包含协议解析、队列、心跳、轮询、超时重试等核心功能
