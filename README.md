Modbus RTU 主站模拟器（基础版）

项目简介

基于 Qt5/C++ 开发的 Modbus RTU 协议主站上位机工具，支持 03（读保持寄存器） 和 06（写单个寄存器） 功能码。可配合虚拟串口与 Modbus Slave 模拟器进行调试，用于工业设备的数据采集与控制。



功能特性

✅ 串口参数配置（波特率、数据位、停止位、校验位）

✅ 手动发送 Modbus 03 读请求，解析并显示寄存器值（十进制/十六进制）

✅ 手动发送 Modbus 06 写请求，修改从站寄存器值

✅ 接收数据以十六进制原始报文显示，便于调试

✅ 记录读写操作日志（时间戳、数据）到表格

✅ 清空表格记录

✅ 基础错误提示（串口打开失败、CRC校验错误等）

技术栈

开发框架：Qt 5.15.2 / Qt 6.x

编程语言：C++17

通信协议：Modbus RTU（03/06 功能码）

串口模块：QSerialPort

构建工具：Qt Creator / MinGW

快速开始

1\. 环境准备

安装 Qt 5.15.2 或更高版本（MinGW 编译器）

安装虚拟串口驱动（推荐 com0com 或 VSPD）

安装 Modbus 从站模拟器（推荐 Modbus Slave）

2\. 测试步骤

使用 VSPD 创建一对虚拟串口，例如 COM3 和 COM4。

打开 Modbus Slave，连接 COM3，设置：

波特率 9600，8 数据位，无校验，1 停止位

从站地址 = 1

在寄存器地址 0 处填入初始值（如 1234）

运行本程序，选择另一个虚拟串口 COM4，相同波特率，点击“打开串口”。

点击“读取寄存器”，界面将显示从站返回的数值。

在“写寄存器”区域输入地址（十六进制）和值（十进制），点击“写入”，观察从站值是否改变。

3\. 编译运行

使用 Qt Creator 打开 ModbusMaster.pro

选择 MinGW 编译器，点击“构建” -> “运行”
